## Objective

The objective of this project is to redesign an existing user interface by applying accessibility principles that improve usability for people with visual, motor, and cognitive impairments.

---

## Tools Used

- Figma
- GitHub
- Microsoft Word
- PDF

---

## Accessibility Improvements

- Improved color contrast
- Larger font sizes
- Better button spacing
- Keyboard-friendly navigation
- Clear labels and placeholders
- Improved icon visibility
- Consistent typography
- Better touch target sizes
- Screen reader-friendly design

## Accessibility Standards Followed

- WCAG 2.1 Guidelines
- Proper Color Contrast
- Readable Typography
- Consistent Navigation
- Accessible Forms

## Project Outcome

The redesigned interface provides a better experience for all users, especially those with disabilities, while maintaining a clean and modern appearance.
